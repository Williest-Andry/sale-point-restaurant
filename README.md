# Restaurant Management
A distributed restaurant management system built with Spring Boot, exposing an independent REST API for handling ingredients, stock movements, prices, dishes, and customer orders.

![Restaurant-img](screenshot/Restaurant-img.png)

### Tech Stack :

Backend: Spring Boot

Database: PostgreSQL

API Design: OpenAPI 3

Testing: Postman, JUnit, Integration testing

### Key Features :
- **Local API :**
CRUD for ingredients, prices, and stock movements

Manage dishes and their composition

Handle orders with dish status transitions

Clean separation: Controller, Service, Repository, Mapper, Configuration
